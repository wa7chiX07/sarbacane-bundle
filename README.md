# sarbacaneBundle

sarbacaneBundle is a Symfony 3.4 bundle for sarbacne .

## config.yml

Use this line in config.yml file.

```yml
sarbacane:
    apiKey: '%apiKey%'
    accountId: '%accountId%'
```

## parameters.yml

```yml
apiKey: [YOUR_APIKEY]
accountId: [YOUR_ACCOUNTID]
```
